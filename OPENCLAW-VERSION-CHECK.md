## 📋 升级步骤

### 方法 1：pnpm 升级（推荐）

```bash
# 1. 查看当前版本
pnpm list -g openclaw

# 2. 升级到最新版本
pnpm add -g openclaw@latest

# 3. 验证升级
openclaw --version

# 4. 检查状态
openclaw status
```

### 方法 2：npm 升级

```bash
# 升级到最新版本
npm install -g openclaw@latest

# 验证
openclaw --version
```

### 方法 3：指定版本升级

